适配器模式 Adapter Pattern - Java实现

定义

将一个类的接口变换成客户端锁期待的另一种接口，从而使原本因接口不匹配而无法工作在一起的两个类能够在一起工作。
也叫作变压器模式，亦称包装模式，但包装模式不止一个。
简单而言，适配器模式就是把一个接口或类转换成其他的接口或类。

应用

优点：

  • 可以让没有任何关系的类在一起运行
  • 增加了类的透明性
  • 提高了类的复用度
  • 灵活度好

注意事项：在详细阶段不要考虑适配器模式，它主要是用来解决正在服役的项目问题
