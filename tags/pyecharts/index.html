绘制漏斗图

我们用pyecharts来绘制漏斗图。

Echarts 是一个由百度开源的数据可视化，凭借着良好的交互性，精巧的图表设计，得到了众多开发者的认可。而 Python 是一门富有表达力的语言，很适合用于数据处理。当数据分析遇上数据可视化时，pyecharts 诞生了。

pyecharts会输出html文件。

默认样式

拿到数据后，我们把数据传给Funnel。

  • 初始配置里设定宽高 InitOpts(width="800px", height="400px")
  • input_data就是我们前面获取的数据列表；将它传给data_pair，即data_pair=input_data
  • 输出的网页文件名为funnel_chart_1.html
  • set_global_opts()方法设置全局配置
    • 标题设置 title="漏斗图1"
    • 副标题设置 subtitle="默认样式"
