OpenAI 在冲击 IPO 的关键冲刺中未能达成关键营收和用户目标

本文最后更新于 2026年4月28日 中午

OpenAI 最近未能实现其新用户和收入目标，这一挫折引发了一些公司领导人的担忧，他们担心公司是否有能力支撑其在数据中心方面的大规模支出。

据知情人士透露，首席财务官 Sarah Friar 已向公司其他领导人表示，她担心如果收入增长不够快，公司可能无法支付未来的计算合同费用。

知情人士透露，董事会成员近几个月来也更加仔细地审查了公司的数据中心交易，并质疑首席执行官萨姆·奥特曼在业务放缓的情况下仍努力获取更多计算能力的做法。

在年底前可能进行的首次公开募股（IPO）之前，严格的支出审查正在限制奥特曼公司曾经无所畏惧的雄心壮志。知情人士透露，弗莱尔和其他高管目前正试图控制成本，并在公司内部建立更严格的纪律，这有时会让他们与首席执行官产生分歧。

多年来，奥特曼一直致力于尽可能多地锁定数据中心容量，他认为计算资源短缺是制约 OpenAI 发展的最大障碍。去年，他进行了一系列交易，使 OpenAI 承担了未来约 6000 亿美元的支出承诺，并将科技行业的成功与 OpenAI 的成功紧密联系起来 。

据知情人士透露，OpenAI 未能实现其内部目标，即在去年底前使其 ChatGPT 的周活跃用户数达到 10 亿。该公司至今仍未正式宣布这一里程碑，这令一些投资者感到不安。知情人士还表示，由于谷歌的 Gemini 在去年底实现了迅猛增长，蚕食了 OpenAI 的市场份额，该公司 ChatGPT 的年度营收目标也未能达成。此外，据知情人士透露，OpenAI 的用户流失率也一直居高不下。

据知情人士透露，OpenAI 今年早些时候在编码和企业市场被 Anthropic 抢占了先机，未能达到多个月度营收目标。

OpenAI 近期完成了硅谷历史上规模最大的一轮融资，筹集了 1220 亿美元，使其财务状况更加稳健。但该公司已签约的计算能力规模庞大，预计在未来三年内（假设其能够实现雄心勃勃的营收目标）就会消耗殆尽 。此外，部分融资还附带条件，取决于与合作伙伴的具体协议。

该公司开发的编码工具 Codex 正迅速普及，同时，它也通过削减其他项目（例如视频生成应用 Sora） 来降低成本。OpenAI 近期发布了 GPT-5.5，这是一款功能强大的模型，在多项行业基准测试中名列前茅。

包括 Anthropic 在内的多家人工智能公司近几周都面临着计算能力不足的问题 ，导致人工智能处理器价格上涨、服务中断和资源配给。这些问题激怒了人工智能产品的高级用户，尤其是程序员，他们发现人工智能系统无法像以往那样完成任务，感到非常沮丧。

OpenAI 在最近一份致投资者的备忘录中表示，其已获得比 Anthropic 更多的计算能力，这使其在用户拓展方面占据优势。这份备忘录已被《华尔街日报》查阅，其中还提及了 Anthropic 首席执行官 Dario Amodei 在近期一次商业会议上对 OpenAI 的隐晦批评。Amodei 当时表示，一些公司在数据中心支出方面“过度冒险”。

OpenAI 的备忘录称：“事后看来，这种谨慎与其说是自律，不如说是低估了需求到来的速度。”

据知情人士透露，近几个月来，弗莱尔也对 OpenAI 计划在今年年底上市表示了保留意见。

她向高管和董事会强调了 OpenAI 需要改进内部控制，并警告说该公司尚未准备好达到上市公司所需的严格报告标准。一些知情人士透露，奥特曼倾向于采取更积极的 IPO 时间表。

OpenAI 在上市前还需解决一系列其他问题。该公司目前正面临领导层真空，其二号人物菲吉·西莫（Fidji Simo）本月初意外请病假。此外，埃隆·马斯克本周提起诉讼， 试图罢免奥特曼（Altman） 并撤销 OpenAI 转型为营利性公司的计划。
