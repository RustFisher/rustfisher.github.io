柱形图

数据会被绘制成垂直，水平或者是层叠效果的柱形图。

注意：

  • 接下来的设置会影响到不同类型的图表。
  • 设置类别可以控制柱形图是垂直或是水平。
  • 使用层叠图形时，需要设置overlap成100。
  • 如果柱形是水平的，x和y坐标会对调过来。
